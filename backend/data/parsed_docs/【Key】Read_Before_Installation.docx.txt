Android Device Service Installation Guide
Overview
Establish connection between your PC and the Android device under the same network, then install the services to the MTR device via the PC.
Enable PC control permission for the device
Locate and enable ADB in the device’s factory menu (Android Debug Bridge: A built-in Android debugging tool. Once enabled, the PC can install apps, transfer files, debug programs, and even modify system settings for the Android device via command line).
Contact the corresponding device after-sales support (MTR Team, IFP Team, LED Team) if you cannot access the factory menu.
Ensure PC and device are on the same network
Connect to the same Wi-Fi or wired network and verify the device IP is reachable via ping.
Install services via PC
Double-click the Install Service tool in the attachment to install the two services.
Note: For Mac users using the tool for the first time, an authorization prompt may pop up. Go to System Settings > Privacy & Security, scroll to the bottom, select "Open Anyway", then double-click the tool again to launch it.
